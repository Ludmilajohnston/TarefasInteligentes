Tarefas Inteligentes - Versão 1

Aplicativo de gerenciamento de tarefas desenvolvido para a disciplina de Programação para Dispositivos Móveis.


Sobre o projeto
O Tarefas Inteligentes é uma aplicação web desenvolvida em formato PWA (Progressive Web App), permitindo ao usuário organizar tarefas do dia a dia de forma simples e prática.
A primeira versão do projeto foi criada com foco nas funcionalidades principais de gerenciamento de tarefas e funcionamento mobile.


Funcionalidades
Adicionar tarefas
Concluir tarefas
Excluir tarefas
Armazenamento local com LocalStorage
Funcionamento offline
Instalação no celular (PWA)
Interface responsiva


Tecnologias utilizadas
HTML5
CSS3
JavaScript
LocalStorage
Service Worker
Manifest.json
PWA

Responsividade
O aplicativo foi desenvolvido para utilização em:
smartphones;
tablets;
desktop.


Funcionalidade Offline
O sistema utiliza Service Worker para permitir funcionamento mesmo sem conexão com internet.


Como executar o projeto
Baixe ou clone o repositório
Abra a pasta do projeto
Execute com Live Server
Acesse pelo navegador


Instalação no celular
Abra o aplicativo no navegador
Clique em:
“Adicionar à tela inicial”
ou “Instalar aplicativo”
O app será instalado no dispositivo


Objetivo da Versão 1
Desenvolver uma aplicação PWA funcional com:


gerenciamento básico de tarefas;
suporte mobile;
armazenamento local;
instalação como aplicativo.
